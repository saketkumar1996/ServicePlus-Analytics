  2. Help
      Screen is divided into three components:
    1. Left Panel: States selection menu
    2. Summary Panel: Key ServicePlus statistics
    3. Right Panel: India map, representing the service score for all states

    The summary panel displays the key statistics for ServicePlus application till date data. For example, data from 14 states, 151 services, 20 departments is available on ServicePlus BI application. The number of applications received and the number of applications processed on the ServicePlus application till date is also shown here. Red-yellow-green indicators help the customer to analyze a state performance from the various aspects.

    The report State Department Service Report right below the Summary Panel, provides a tabular list of the State Department Service wise Average Service Score. The average service score for a state is derived by rating a service on multiple Key Performance Indicators (KPIs). The average service score for a state is derived by taking the average score for all services in the state.

    The India map displays the state wise service performance score. The colours of the states denote the relative performance of a state with respect to the average score of all states. On mouse-over, the Service Performance Score for the selected state is shown.

  3. KPI(Key Performance Indicator)

    KPI is a measurable value that demonstrates how effectively an organization is achieving key business objectives. Organizations use KPIs at multiple levels to evaluate their success at reaching targets.

    KPI are measured at service level. All the KPIs are independent and can be applied to all the services/departments and states to evaluate the success of an organization or of a particular activity in which it engages.

    1. No of submissions channels supported for a service
    2. Percentage of applications which are not processed(No action taken)
    3. Average number of times supplementary documents requested
    4. Percentage of applications Processed Within SLA
    5. Percentage of applications Processed Beyond SLA
    6. Percentage of applications delivered in 1-7 days post SLA
    7. Percentage of applications delivered in 8-14 days post SLA
    8. Percentage of applications delivered in 15-30 days post SLA
    9. Percentage of applications delivered in 30 days post SLA
    10. Percentage of applications delivered in greater than 31 days post SLA
    11. Average processing time vs SLA defined
    12. Percentage of applications put on hold
    13. Average hold time for processed applications
    14. Percentage of total applications called back
    15. Average no. of times application called back
    16. Percentage of total applications called back after delivery
    17. Percentage of total applications called back before delivery
    18. Percentage of applications Rejected
    19. Percentage of applications revived after rejection
    20. Average time for submission of application in the system - data entry operators
    21. Average time for submission of application in the system - kiosk
    22. Applications received online
    23. Payment received online
    24. Percentage of applications with digital signature used for output certificate
    25. Application processed per employee per day
